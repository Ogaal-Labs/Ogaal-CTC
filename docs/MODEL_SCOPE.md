# Model Scope

## Release Position

`Ogaal CTC` is a Somali automatic speech recognition system released by Ogaal Labs.

## Intended Use

- Somali speech transcription
- local CLI transcription
- local browser demo transcription
- developer integration for Somali voice products

## Training Intent

This system was intentionally trained and packaged for Somali speech recognition.

English was not part of the training objective for this release.

## Public Description

- a Somali XLSR/CTC ASR model
- released together with its published LM decode path
- intended for Somali transcription workflows

## Data Story

The release includes a private Ogaal Labs collection effort built around roughly 5,000 curated prompts recorded by 19 speakers across varied genders, accents, and speaking styles.
